Nexus Gaming – Business Portfolio Website
Project Overview

Nexus Gaming is a business portfolio website designed to showcase gaming accessories using an e-commerce style interface.
The project is developed as part of the Web Technology Lab and demonstrates both front-end and basic back-end web development concepts.

This website focuses on presenting products, simulating cart functionality, handling contact form submissions, and providing an admin panel for viewing user messages.
It does not support real product ordering, online payments, or gameplay.

Objectives of the Project

To design a responsive and visually appealing business portfolio website

To demonstrate an e-commerce style layout for showcasing products

To implement client-side interactivity using JavaScript

To handle contact form submissions using a Node.js backend

To provide an admin panel for viewing submitted messages

Features

Gaming-themed user interface

Product listing and cart simulation

Contact form with server-side message storage

Date and time logging for contact submissions

Password-protected admin panel

Admin dashboard to view submitted messages

Organized project structure for easy understanding

Technologies Used

HTML5 – Structure of web pages

CSS3 – Styling and layout

JavaScript – Client-side interactivity

Node.js – Backend processing

Express.js – Server framework

File System (fs module) – Message storage

Important Notes

This project is created for academic and learning purposes only.

The e-commerce functionality is simulated and does not involve real transactions.

The website is not an online gaming platform and does not allow users to play games.

Backend functionality is intended for demonstration and understanding of server-side concepts.

Conclusion

The Nexus Gaming business portfolio website successfully demonstrates the use of front-end and back-end technologies to build an interactive web application.
The project helps in understanding website structure, styling, client-side scripting, backend handling, and basic admin functionality, making it suitable for academic evaluation in Web Technology Lab.
